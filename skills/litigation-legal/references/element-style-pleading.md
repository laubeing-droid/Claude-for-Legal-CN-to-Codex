## 二、通用要素式格式（提取自 67 类共性结构）

### 起诉状强制要素

| 区块 | 必填项 |
|:-----|:-----|
| 当事人信息 | 姓名/名称、住所地、法定代表人/负责人、统一社会信用代码、联系电话 |
| 诉讼请求 | 逐条编号、金额精确到分、附计算方式 |
| 事实与理由 | 时间线分段（每段 ≤5 行）、附关键证据引用 |
| 法律依据 | 法律名称 + 具体条款 + 适用理由（表格格式） |
| 证据清单 | 证据名称 + 份数 + 证明目的 |

### 答辩状强制要素

| 区块 | 必填项 |
|:-----|:-----|
| 对原告诉讼请求的逐项回应 | 承认/否认/部分承认 + 理由 |
| 核心抗辩 | 逐条列出 + 法条依据 |
| 证据清单 | 对应每项抗辩的证据 |

### AI 生成约束

- ⛔ 禁止美式 Pleading 叙述体（长段落背景陈述）
- ⛔ 禁止"Whereas""Hereby""In Witness Whereof"等翻译腔
- ✅ 必须使用表格 + 要素填空格式
- ✅ 当事人信息必须包含"法定代表人"字段（公司法第11条）
- ✅ 金额必须有计算公式（本金 + LPR×N倍×天数）
- ✅ 证据清单必须附在文末（证据名称 + 证明目的二列）
