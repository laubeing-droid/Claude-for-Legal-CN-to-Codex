# 民商事要素式诉状模板集（12 案由）

> 来源：最高人民法院《民事起诉状示范文本》民商事类
> 以下每个案由的要素清单均为 SPC 官方模板原文提取。

---

## 一、房屋买卖合同纠纷

### 诉讼请求
| # | 要素 |
|:--|:-----|
| 1 | 交付房屋 |
| 2 | 协助办理过户登记 |
| 3 | 支付逾期交房违约金 |
| 4 | 支付逾期过户违约金 |
| 5 | 支付逾期付款违约金 |
| 6 | 是否主张解除合同 |
| 7 | 是否主张双倍返还定金/赔偿损失 |
| 8 | 标的总额 |

### 事实与理由要素
合同签订→购房款支付→房屋交付→过户登记→违约事实（逾期交房/逾期过户/逾期付款/一房二卖/面积差异/质量问题）

## 三、建设工程施工合同纠纷

### 诉讼请求
| # | 要素 |
|:--|:-----|
| 1 | 支付工程款（元） |
| 2 | 迟延支付工程款的利息/违约金 |
| 3 | 是否主张建设工程价款优先受偿权 |
| 4 | 是否要求发包人/转包方/分包方承担责任 |
| 5 | 是否要求赔偿损失（停窝工损失/其他） |
| 6 | 是否退还超付的工程款 |
| 7 | 是否支付超付工程款的利息 |
| 8 | 是否承担修复责任（修复/付修复费/减工程款） |

### 事实与理由要素（15项）
合同签订→签订主体→工程进度款支付→工程量/工程质量→违约责任→实际开工/竣工日期→工期→工程款支付→垫资→工程价款优先受偿权→竣工结算→保修义务→质量争议→转包/分包/挂靠→证据清单

## 四、金融借款合同纠纷

| 1 | 借款本金（元） |

| 4 | 是否主张实现债权的费用 |
| 5 | 标的总额 |

### 特有要素

- 担保合同签订情况（抵押/质押/保证）
- 不动产抵押登记情况
- 借款发放/偿还情况
