# WorkBuddy 中国法务 Skill 压缩包

这个仓库放的是已经适配 WorkBuddy 的中国法务 skill 压缩包。

下载仓库后，打开 `可导入技能包` 文件夹，里面每个压缩包都是一个可以上传到 WorkBuddy 的 skill。文件名已经改成中文，按名字选择需要的即可。

## 仓库内容

- `可导入技能包`：可直接上传到 WorkBuddy 的 skill 压缩包。
- `LICENSE`：原项目许可证。
- `NOTICE`：来源和改编说明。

目前共有 151 个 skill，覆盖合同审查、数据合规、人工智能治理、产品合规、公司治理、劳动用工、知识产权、争议解决、监管合规、法律援助和中国法学习等方向。

## 来源

这些 skill 的原始素材来自 Anthropic PBC 发布的开源项目 `Claude for Legal`。原项目发布在 GitHub，内容是面向 Claude 的法律工作流和 skill 示例，覆盖商事合同、公司治理、隐私与数据、人工智能治理、知识产权、劳动用工、争议解决、监管合规、法律援助和法律学习等场景。

https://github.com/anthropics/claude-for-legal

原项目版权归 Anthropic PBC 所有，许可证为 Apache License 2.0。本仓库基于该开源项目做 WorkBuddy 中国法务场景适配，保留了原项目的 `LICENSE` 和 `NOTICE` 文件，并在 `NOTICE` 中说明了改编来源。

本仓库的改编重点是把原始法律类 skill 整理成可以上传到 WorkBuddy 的中文压缩包，同时补充中国大陆法律和监管语境下的使用提示。使用时请结合具体事实、业务场景和现行法律法规进行专业复核。

## 做了哪些适配

- 把 skill 名称改成中文，方便在 WorkBuddy 里识别。
- 默认按中国大陆法律和监管语境工作。
- 默认使用简体中文输出。
- 增加中国法务常用的风险提示和人工复核提醒。
- 按 WorkBuddy 上传要求重新打包，每个压缩包都可以单独导入。

## 怎么用

1. 下载这个仓库并解压。
2. 打开 `可导入技能包` 文件夹。
3. 打开 WorkBuddy，进入“技能”。
4. 点击“添加技能”，选择“上传技能”。
5. 选择一个中文命名的压缩包。
6. 导入完成后，在新任务里选择刚导入的 skill 使用。

导入时可以对照下面两张图：

![点击添加技能](docs/images/import-flow/02-点击添加技能.png)

![选择中文压缩包](docs/images/import-flow/04-选择中文压缩包.png)

建议先导入一个常用 skill 试试，例如：

- `商事合同法务-保密协议审查`
- `数据合规与个人信息保护-个人信息保护影响评估`
- `产品合规法务-上线审查`
- `劳动用工法务-解除－终止劳动关系审查`

## 使用提醒

这些 skill 适合做法律研究、内部合规分析、材料整理、草稿生成和学习训练。正式发送、提交、签署或对外使用前，请让执业律师、公司法务负责人或相应专业人员复核。
